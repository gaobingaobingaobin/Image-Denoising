FOE Installation help:

- Download link: 'http://www.visinf.tu-darmstadt.de/vi_research/code/foe.en.jsp'
- Set your location of reference images and the script file 'foe_denoise_demo.m' inside the file 'method_params.txt'.
- Make sure to convert 'string' to 'double' format for the numrical paramter imported from the parameter file. For eg: sigmaDouble = str2double(sigma); 
- You can set the vaules of sigma(Parameter for noise level) and the type of model from 'method_params.txt'.
